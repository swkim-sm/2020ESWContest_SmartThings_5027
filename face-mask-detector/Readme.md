개발 환경 세팅  
======  

1. 아나콘다 프롬프트 or CMD 실행  

2. (pip 패키지 업그레이드)  
`conda upgrade pip`  
`pip install upgrade`  

3. 가상환경 새로 설치  
`conda create -n (env) python=3.7  
activate (env)  `  

4. tensorflow 설치  
`pip install tensorflow==1.15.2  `  

5. 버전 확인  
`python  
import tensorflow as tf  
tf.__version__  `  

6. 라이브러리 설치  
` pip install numpy matplotlib pillow opencv-python  `
